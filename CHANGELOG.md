# 桌面备忘录 - 更新日志

## 版本 1.1.0

### 云同步功能

#### Supabase 云端同步
- **用户认证** - 支持邮箱注册和登录
- **数据同步** - 备忘录、配置、垃圾桶数据全部支持云端同步
- **登录状态持久化** - 登录后自动保持登录状态，无需重复登录
- **安全存储** - 敏感信息使用 electron-store 加密存储
- **行级安全** - Supabase RLS 确保用户只能访问自己的数据

#### 同步操作
- **上传所有数据** - 一键上传本地数据到云端
- **下载所有数据** - 一键从云端下载数据到本地（会提示确认）
- **数据统计反馈** - 上传/下载完成后显示数据条数统计
- **操作状态提示** - 实时显示操作进度和结果

#### 界面优化
- **云同步设置面板** - 新增独立的云同步设置页面
- **滚动条样式统一** - 云同步页面滚动条与其他页面风格一致
- **按钮状态反馈** - 上传/下载时按钮禁用并变淡，防止重复点击

### 使用说明

1. 在 Supabase 创建项目并获取 URL 和 Anon Key
2. 在应用设置中点击「云同步」
3. 输入 Supabase URL 和 Anon Key 保存配置
4. 注册/登录账号
5. 点击「上传所有数据」或「下载所有数据」进行同步

## 版本 1.0.0

### 核心功能

#### 备忘录管理
- **添加备忘录** - 点击「+ 添加备忘」按钮或按 Enter 键创建新备忘
- **编辑备忘录** - 直接点击文本区域进行编辑
- **删除备忘录** - 点击右侧 × 按钮，移入垃圾桶
- **完成状态** - 点击左侧复选框切换完成/未完成状态
- **优先级设置** - 三种优先级（高/中/低），按颜色区分排序
- **创建时间显示** - 每条备忘录右下方显示创建时间（精确到分钟）
- **智能排序** - 正在编辑的备忘录临时置顶，失去焦点后按优先级排序

#### 垃圾桶功能
- **回收站机制** - 删除的备忘录移入垃圾桶，按日期分类
- **日期分组** - 垃圾桶按日期（年/月/日）以文件夹形式分类显示
- **时间信息** - 每条记录显示创建时间和删除时间
- **批量删除** - 勾选多条记录后，永久删除
- **空状态提示** - 垃圾桶为空时显示提示信息

#### 窗口管理
- **无边框窗口** - 透明无边框设计
- **窗口置顶** - 可设置窗口始终在最上层
- **背景透明度** - 可调节窗口背景透明度（30%-100%）
- **窗口位置记忆** - 关闭后记住窗口位置和大小
- **系统托盘** - 最小化到系统托盘，双击托盘图标显示/隐藏窗口
- **隐藏任务栏图标** - 窗口不在任务栏显示，只在系统托盘显示

#### 快捷键功能
- **全局快捷键** - 默认 `Ctrl + Shift + M` 显示/隐藏窗口
- **自定义快捷键** - 在设置中可自定义快捷键组合
- **快捷键重置** - 一键重置为默认快捷键

#### 系统功能
- **开机自启动** - 可设置开机自动运行
- **单实例模式** - 防止多开，再次点击 exe 唤出已有实例
- **数据持久化** - 数据保存在本地 JSON 文件

### 界面优化

#### 设置面板
- **滚动条支持** - 设置内容过多时可滚动
- **垃圾桶按钮** - 金黄色按钮（#FFE69A）
- **快捷键输入** - 点击输入框后按下组合键即可设置

- **Header 透明度** - 标题栏透明度随背景透明度联动变化
- **拖拽区域** - 标题栏可拖拽移动窗口，按钮区域可点击

#### 交互优化
- **ESC 键退出** - 按 ESC 键可退出当前页面（设置面板、垃圾桶等）
- **返回按钮** - 垃圾桶页面左上角返回按钮
- **确认对话框** - 清空所有、永久删除等操作有确认提示

### 技术实现

#### 架构设计
- **Electron 28** - 跨平台桌面应用框架
- **主进程/渲染进程分离** - 安全的 IPC 通信
- **contextBridge** - 安全暴露 API 给渲染进程
- **JSON 文件存储** - 本地数据持久化

#### 安全特性
- **nodeIntegration: false** - 禁用 Node.js 集成
- **contextIsolation: true** - 启用上下文隔离
- **preload 脚本** - 安全的 API 暴露方式

#### 单实例实现
- **requestSingleInstanceLock** - 单实例锁
- **app.exit(0)** - 强制立即退出第二个实例
- **second-instance 事件** - 唤出已有实例窗口
